# refcode/ 目录说明（参考代码与示例）

## HEADER
- **Purpose**: 保存可复用的代码示例、模式实现和第三方集成样例，作为开发参考
- **Status**: Active
- **Date**: 2025-11-14
- **Dependencies**: 相关模块源码
- **Target**: 开发者与代码生成类 AI 助手

## 内容说明

`refcode/` 适合存放：

- 经过验证的代码片段与最佳实践实现
- 对第三方库或服务的示例集成
- 可以被复制粘贴并稍作修改即可使用的模板代码

这些内容可以被 AI 助手当作「模式库」，在生成新代码时优先遵循这里的风格与约定。
