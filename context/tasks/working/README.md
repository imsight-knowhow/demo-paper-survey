# tasks/working/ 目录说明（进行中的任务）

## HEADER
- **Purpose**: 记录当前正在进行的任务详情，便于跟踪与接力
- **Status**: Active
- **Date**: 2025-11-14
- **Dependencies**: tasks/ 总体约定
- **Target**: 当前负责该任务的开发者与 AI 助手

## 内容说明

将每个「正在做」的任务以单独文件形式放在此处，例如：

- `task-implement-user-authentication.md`
- `task-fix-memory-leak.md`

任务完成后，应移动到 `tasks/done/` 目录，以保持此目录只反映当前的活动工作。
